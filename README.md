# Candle Grove Home Care Website

Quick start:
1. Install Node.js
2. Open this folder in a terminal
3. Run:
   npm install
   npm run dev

Main file to edit:
- app/page.js

Replace these placeholders:
- phoneDisplay
- phoneHref
- email
- referralEmail
- careersEmail
- serviceArea
- brochureUrl
- clientFormAction
- referralFormAction
- caregiverFormAction
- primaryDomain

Recommended hosting:
- Vercel
